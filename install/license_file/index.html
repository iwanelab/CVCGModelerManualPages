    ライセンスファイルの追加¶

    本アドオンを動作させるためには、有効なライセンスファイルが必要です。ここでは、ライセンスファイルの追加方法について解説します。

    CVCGModelerServiceの起動¶

    ライセンスファイルはCVCGModelerServiceへと追加します。そこで、まだCVCGModelerServiceが起動していなければ、スタートメニューから起動してください。

    Note

    CVCGModelerServiceは、タスクトレイ常駐型のアプリケーションです。そのため、スタートメニューから起動しても CVCGModelerServiceのウィンドウが開くわけではありません。

    設定¶

    手順¶

    STEP 1¶

    タスクトレイからCVCGModelerServiceのアイコンを右クリックし、メニューを開きます。

    STEP 2¶

    メニューから「settings」を選択して、設定画面を開きます。

    STEP 3¶

    インストール直後はライセンスファイルが空の状態なので、次の画面のように、Not Found と表示されます。ライセンスファイルを追加するために、Openを選択します。

    STEP 4¶

    ファイル選択ダイアログが表示されるので、適切なライセンスファイルを選択して、開く を選択します。

    結果¶

    適切なライセンスファイルが設定されていれば、State表示がAuthenticatedとなるはずです。StateがAuthenticated以外の表示の場合、CVCGModelerService は動作しません。
