    CV動画操作¶

    ここではCV動画の操作について説明します。

    基本操作¶

    マウスでの操作¶
    中ボタンドラッグ 映像を回転します。
    マウスホイール奥 映像を拡大します。
    マウスホイール手前 映像を縮小します。

    フレーム操作¶

    CV動画のフレーム操作は、Blenderのフレーム操作機能・UIを使用して行います。

    以下で説明する操作方法については、Blender標準のものです。

    キーボードでの操作¶
    スペース 動画を再生・停止します。
    右キー フレームをひとつ次へ進めます。
    左キー フレームをひとつ前へ戻します。
    Shift + 右キー 最終のフレームへ移動します。
    Shift + 左キー 最初のフレームへ移動します。

    Blender 画面からの操作¶
    ① 現在表示しているフレームを示します。ドラッグすることでフレーム移動が可能です。移動したいフレーム周辺をクリックすることで移動も可能です。
    ② 動画を逆再生します。
    ③ 動画を再生します。
    ④ 最初のフレームへ移動します。
    ⑤ 最終のフレームへ移動します。
    ⑥ 現在表示しているフレームを示します。直接フレーム番号を入力して移動することもできます。
    ⑦ 最終フレーム番号を示します。

    CV動画選択¶

    CV File List には現在参照しているCV動画ファイルの一覧が表示されます。この中から、Blender上で表示したい動画をクリックして選択することで、使用する動画を変更することができます。

    カメラ選択¶

    Camera Index から、使用カメラを切り替えることができます。

    フレーム情報¶

    フレームのカメラ位置に関する座標値を確認することができます。
