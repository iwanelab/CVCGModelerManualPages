    オリエンテーション作成¶

    ここではオリエンテーション作成について説明します。

    オリエンテーションとは¶

    Blenderには Transform Orientation という枠組みがあります1。 適切な Transform Orientation を使うことで、この座標系の軸に沿ったオブジェクトの操作が可能となります。

    たとえば、軸に沿ってオブジェクトを移動したり、頂点・辺・面を押しだしたり、様々な操作を効率的に行うことができます。

    このオリエンテーションを、

    1. 見えている画像から自動的に計算
    2. 3次元計測によって作成された辺から作成

    することが可能です。

    自動でのオリエンテーション計算¶

    Blender上に見えているCV映像から、それに合わせたオリエンテーションを作成します。

    手順¶

    STEP 1¶

    オリエンテーションの作成対象となるものが見えるように、カメラの向きと位置を調整します。

    Tip

    できるだけ対象物のみが映るように調整した方がいい結果が得られます。

    Warning

    基準となる縦横のラインが取れない映像では失敗します。

    STEP 2¶

    CVメニューから Fit coordinate system を選択します。

    結果¶

    Success

    オリエンテーション作成に成功した場合、Info表示部分に成功を表すメッセージが表示されます。

    手動でのオリエンテーション作成¶

    自動でうまくオリエンテーションが計算できないときなど、手動でオリエンテーションを作成することも可能です。

    手順¶

    STEP 1¶

    まず、オリエンテーションを作成したい方向の辺を作成します。

    Tip

    特徴的な2点をエピポーラ計測で作成します。

    頂点を作成した後、Blenderで辺を作る方法については以下を参照のこと。

    https://docs.blender.org/manual/en/2.83/modeling/meshes/editing/vertex/make_face_edge.html

    STEP 2¶

    鉛直方向の辺を選択します。

    STEP 3¶

    CVメニューから Set vertical を選択します。

    Note

    一度 Set vertical を行えば、以降の Set rotation の実行では、この鉛直軸が使われます。

    Note

    この時点で新たなオリエンテーションが作成されています。鉛直方向の軸のみが合っているオリエンテーションです。

    STEP 4¶

    回転方向の辺を選択します。

    STEP 5¶

    CVメニューから Set rotation を選択します。

    Note

    一度 Set rotation を行えば、以降の Set vertical の実行では、この回転方向軸がオリエンテーション計算に使われます。

    1. https://docs.blender.org/manual/en/latest/editors/3dview/controls/orientation.html ↩
